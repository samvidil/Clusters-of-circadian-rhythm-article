README

Analyses were undertaken in R version 4.2.2 

- In R all th next packages are needed to reproduce the code: haven, corrplot, tidyverse, nnet, questionr, factoextra, car, parallelDist, dplyr, lubridate, multcomp
The command install.packages("...") can install its.

- To extract the circadian rhythm metrics used, GGIR 3.1-7 was used. Available in here: https://zenodo.org/records/14218292

GGIR extraction are given by GGIR_extraction_WII and GGIR_extraction_UKB.
R analyses are given by Clusters_WII and Clusters_UKB.
